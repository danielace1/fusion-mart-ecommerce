## ğŸ“¦ Installation & Setup

### Prerequisites

Ensure you have the following installed:

- Node.js
- MongoDB
- Redis

### 1. Clone the Repository

```bash
  git clone https://github.com/danielace1/fusion-mart.git
  cd fusion-mart
```

### 2. Backend Setup

```bash
  cd backend
  npm install
  npm start
```

### 3. Frontend Setup

```bash
  cd frontend
  npm install
  npm start
```
